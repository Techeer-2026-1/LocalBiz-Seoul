## 1. 요구사항

skill_router.sh와 intent_gate.sh의 트리거 키워드가 한국어 어미 변형을 커버하지 못해 "최신화해줘", "정리하자", "추가하자", "검토해줘", "수정해", "바꿔줘" 등의 프롬프트가 훅을 우회함. 어간 기반 매칭으로 전환하여 커버리지 확대.

### 우회 사례 (실제 발생)
- "최신화해줘" → 매칭 없음 → plan 없이 직접 코드 수정
- "정리좀하자" → 매칭 없음 → plan 없이 파일 이동
- "추가하자" → "추가해줘"는 있지만 "추가하자"는 없음
- "검토해줘" / "바꿀 거 있으면" / "수정해줘" → 매칭 없음
