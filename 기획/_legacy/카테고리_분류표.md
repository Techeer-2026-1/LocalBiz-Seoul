# LocalBiz Intelligence — 카테고리 분류표

**버전**: v0.2
**작성일**: 2026-04-11 (v0.1) / **2026-04-12 (v0.2 bump)**
**작성**: 이정 (PM/BE) + 메인 Claude (plan #11 초안 → plan #12 승격)
**상태**: **v0.2 정식 적용** (ETL 강제 함수: `backend/scripts/etl/validate_category.py`)

## 권위

- ERD v6.1/v6.2 §4.1 places 테이블 명세
- 본 문서 (카테고리 enum + 매핑표)
- `profile_report.md` (2026-04-10, 153 CSV 전수 프로파일링)
- ETL은 본 문서 + `validate_category()` 함수로 강제

## 적용 대상

- `places.category` (대분류, VARCHAR(50) NOT NULL)
- `places.sub_category` (소분류, VARCHAR(100) NULL)
- 모든 ETL (현재 legacy `etl_places.py` + 신규 ETL)
- 신규 csv 적재 시 본 분류표를 따라야 함
