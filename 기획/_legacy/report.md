# 이미지 기반 장소 추천 파이프라인 설계 논의 정리
> 작성 기준: 2026-04-01

---

## 1. 서비스 개요

서울 지역 기반 만능 자비스 서비스. 사용자가 사진을 업로드하면 유사한 분위기의 장소를 추천해주는 기능을 포함한다. 이미지 검색 파이프라인은 전체 서비스의 일부 기능.

---

## 2. 기능 범위 결론

### 장소 식별 ("여기 어디야?")
- OCR 기반 간판 인식은 간판이 잘 찍혔을 때만 유효
- 일반 식당/카페의 정확한 식별은 현실적으로 어려움
- **우선순위 낮음 → 식별 실패 시 유사 추천으로 자연스럽게 fallback**

### 유사 장소 추천 ("이런 분위기 추천해줘") ✅ 메인
- CLIP은 Phase 3으로 연기
- Gemini Vision + Gemini 임베딩 조합으로 진행

---

## 3. 유사 장소 추천 파이프라인 (최종)

```
사용자 사진 업로드
    ↓
[Step 1] Gemini Vision → 텍스트 키워드 추출
    ("아늑한 분위기, 우드톤 인테리어, 조용한 카페")
    ↓
[Step 2] Gemini 임베딩 모델 → 벡터 변환
    ↓
[Step 3] OpenSearch k-NN → DB 장소 벡터와 유사도 비교
    ↓
Top 5 유사 장소 반환
```

> ⚠️ **핵심 전제**: DB 장소 텍스트도 반드시 동일한 Gemini 임베딩 모델로 벡터 저장해야 함. 다른 모델 사용 시 벡터 공간 불일치로 비교 불가. 비정형 데이터 팀과 반드시 확인 필요.

---

## 4. 비정형 데이터 수집 파이프라인 (Phase 1)

### 4-1. 전체 흐름

```
API/크롤링으로 텍스트 수집
    ↓
S3에 원본 텍스트 저장
(로컬 저장 불가 — 서울 전체 수백만 건)
    ↓
청킹 (긴 텍스트를 적당한 크기로 분할)
    ↓
Gemini 임베딩 모델로 벡터 변환 (배치 API 활용)
    ↓
OpenSearch에 벡터 저장
```

### 4-2. 데이터 소스별 정리

| 소스 | 제공 데이터 | 비용 | 비고 |
|------|-----------|------|------|
| Google Places API | 리뷰 텍스트 (장소당 최대 5개) | 유료 (월 $200 무료) | 리뷰 짧게 잘림 |
| 네이버 검색 API | 블로그 URL 목록 | 무료 (일 25,000건) | 본문은 별도 크롤링 필요 |
| TourAPI | 관광지 설명 텍스트 | 무료 | 카페/음식점 없음 |
| 카카오맵 API | 장소 기본 정보 | 무료 | 리뷰 텍스트 미제공 |
| 공공데이터포털 | 병원/화장실/시설 등 | 무료 | 위치 정보만, 키워드 없음 |

### 4-3. 장소 타입별 데이터 전략

- **카페/음식점/숙박/쇼핑** → Google Places 리뷰 + 네이버 블로그 크롤링 병행
- **관광지/랜드마크** → TourAPI 설명 텍스트 활용
- **화장실/경찰서/소방서/학교 등** → 공공데이터포털로 충분 (분위기 추천 불필요, 위치 정보만 있으면 됨)

### 4-4. 청킹 전략

블로그 글 전체를 하나의 벡터로 만들면 너무 커서 검색 정확도가 낮아짐. 적당한 크기로 잘라서 각각 벡터로 변환.

```
[블로그 글 전체]
"성수 블루보틀 다녀왔어요. 인테리어가 미니멀하고 조용해서
작업하기 좋아요. 커피도 맛있고..."

↓ 청킹 (200~500자 단위)

chunk 1: "성수 블루보틀 다녀왔어요. 인테리어가 미니멀하고..."
chunk 2: "조용해서 작업하기 좋아요. 커피도 맛있고..."

↓ 각각 Gemini 임베딩 → OpenSearch 저장
```

## 5. 역할 분담

| 팀 | 담당 내용 |
|------|---------|
| 비정형 데이터 팀 | 크롤링/API 수집 → S3 저장 → Gemini 임베딩 변환 → OpenSearch 적재 |
| 이미지 파이프라인 팀 | 사진 → Gemini Vision → 임베딩 → OpenSearch 유사도 검색 → Top 5 반환 |

---

## 6. 기술 스택

| 기술 | 용도 | 비용 |
|------|------|------|
| Gemini Vision API | 사진 → 텍스트 키워드 추출 | 유료 (무료 한도 활용) |
| Gemini 임베딩 모델 | 텍스트 → 벡터 변환 | 유료 (무료 한도 활용) |
| OpenSearch k-NN | 벡터 유사도 검색 | 무료 (자체 운영) |
| Google Places API | 리뷰 수집, 장소 2차 검색 | 유료 (월 $200 무료) |
| TourAPI | 관광지 텍스트 데이터 | 무료 |
| AWS S3 | 원본 텍스트 저장 | 유료 (소액) |
| PostGIS | GPS 기반 거리 필터 | 무료 |
